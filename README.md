## Troubleshooting
### Evans CLI

- Download : https://github.com/ktr0731/evans/releases - evans_windows_386.tar.gz

- Run Evans command in `cmd`
    - `evans -r -p 50051`
    -  `show package`
    - `show service`
    - `show message`
    - `package <packageName>`
    - `service <serviceName>`
    - `call <MethodName : eg. SumService>`
